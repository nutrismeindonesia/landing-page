NUTRISME LANDING PAGE
=====================

File utama:
- index.html
- style.css
- script.js
- assets/

Cara membuka:
1. Klik dua kali index.html, atau
2. Jalankan local server dari folder ini, misalnya:
   python3 -m http.server 8000
   lalu buka http://localhost:8000

Catatan formulir:
- Nama Lengkap, Alamat, No. Handphone, dan checkbox Privacy & Policy wajib diisi.
- No. Handphone hanya menerima angka (8-15 digit).
- Tombol Submit baru aktif setelah seluruh data valid.
- Versi ini adalah preview front-end. Data belum dikirim ke server/WhatsApp/email.
  Sambungkan handler submit di script.js ke kanal pemesanan yang diinginkan.
